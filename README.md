# 🎯 Quiz Web Application (Course Team Project)

This project is a **team-based quiz web application** developed as part of our coursework.  
It allows users to test their knowledge in **HTML, CSS, and JavaScript** through an interactive and modern interface.

## 📌 Project Overview

The application provides a smooth quiz experience where users can:
- Start the quiz from a login/start page  
- Select a programming language  
- Answer multiple-choice questions  
- View their performance at the end  

---

## 🚀 Features

- 🔐 Start/Login Page  
- 🌐 Language Selection (HTML, CSS, JavaScript)  
- ❓ Multiple Choice Questions  
- 🎨 Modern UI Design  
- ⚡ Dynamic Question Rendering using JavaScript  
- 📊 Score Calculation  

---

## 🖥️ Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling and layout  
- **JavaScript** – Logic and interactivity
